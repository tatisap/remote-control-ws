# Remote control

## Installing

Clone this repo to your local machine. Checkout to dev branch. Then run from project root:

```bash
  npm install
```

## Running

You can specify ```HTTP_PORT``` and ```WS_PORT``` in .env file.

To run app use:

```bash
  npm run start
```

Then launch app in browser (```http://localhost:{HTTP_PORT}```);

Enjoy!
